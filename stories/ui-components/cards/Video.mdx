# Video card:
- This component adds the [video player](/docs/ui-components-video-player--video-player) component into a [card](/docs/ui-components-cards-default--default-card).

## Copy/paste the correct bits of HTML
- To use this component, simply copy/paste this fragment of the HTML from the snippet.
```
<article class="ui-card ui-video-card ui-card--${args.cardOrientation}">
    ...
</article>
```
- Please see [how to use snippets](/docs/html-snippets-snippets-how-to-use-snippet-html--page) for more information about removing `${args.}` text from snippet HTML.
