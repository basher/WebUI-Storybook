# Skiplinks
- Skiplinks are hidden visually, and normally only accessible via keyboard `:focus`.

## How to test
- Click the `'Open canvas in new tab'` button in the Storybook `Canvas mode` toolbar.
- Then use the keyboard `TAB` key to navigate the skip links, which will be shown on keyboard ``:focus` only.
